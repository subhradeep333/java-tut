<h1>🤝 Contributing to Java Repository</h1>
<p>🙏 Thank you for contributing! Please follow these guidelines.</p>

<h2>📌 Contribution Rules</h2>
<ul>
  <li>💻 Write clean and readable Java code</li>
  <li>🔤 Follow proper naming conventions</li>
  <li>📝 Add comments where needed</li>
  <li>✅ Ensure code is tested and error-free</li>
</ul>

<h2>📁 Folder Structure</h2>
<ul>
  <li>📂 Place files in correct folders (Encapsulation, Inheritance, etc.)</li>
  <li>🏷️ Use meaningful file names</li>
</ul>

<h2>✍️ Code Guidelines</h2>
<ul>
  <li>📏 Use proper indentation</li>
  <li>🔠 Class names → PascalCase</li>
  <li>🔡 Variables/methods → camelCase</li>
</ul>

<h2>🚀 How to Contribute</h2>
<pre>
git checkout -b feature/your-feature-name
git commit -m "Added: description"
git push origin feature/your-feature-name
</pre>

<h2>❌ What Will Be Rejected</h2>
<ul>
  <li>🚫 Poorly formatted code</li>
  <li>🔁 Duplicate or irrelevant content</li>
  <li>⚠️ Plagiarized submissions</li>
</ul>

<h2>📬 Need Help?</h2>
<p>💡 Open an issue for discussion.</p>
